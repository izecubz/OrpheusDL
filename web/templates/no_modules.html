OrpheusDL requires at least one module to be installed to function properly.

Modules are the components that allow OrpheusDL to interact with different music services.

How to Install Modules

To install modules, follow these steps:

  1. Create a modules directory

    Make sure you have a modules directory in your OrpheusDL installation.

    mkdir -p modules
  2. Clone module repositories

    Clone the module repositories into the modules directory. For example:

    cd modules
git clone https://github.com/your-username/your-module.git
  3. Install module dependencies

    Some modules may require additional dependencies. Check the module's documentation for details.

  4. Configure modules

    Create a config/modules/your-module/settings.json file with the necessary configuration.
